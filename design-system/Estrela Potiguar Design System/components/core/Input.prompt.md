Inset text field, used most often as the map search bar. Focus shows the star-yellow ring.

```jsx
<Input placeholder="Buscar praias, fortes, dunas…" leadingIcon={<SearchIcon/>} />
<Input size="lg" invalid trailingIcon={<XIcon/>} />
```

Props: `size`, `leadingIcon`, `trailingIcon`, `invalid`, `fullWidth` (default true).
