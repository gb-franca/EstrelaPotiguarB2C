Status / metadata pill for attraction state and labels.

```jsx
<Badge tone="success" dot>Aberto agora</Badge>
<Badge tone="star">Top 10</Badge>
<Badge tone="aurora" solid>Destaque</Badge>
<Badge tone="danger" dot>Fechado</Badge>
```

Tones: `neutral | star | aurora | mare | success | danger | info`. Use `solid` for emphasis, `dot` for live status.
